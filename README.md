escoops
=======

Run a command when ESC key is hit

Usage
-----

```
$ escoops 'tw --yes Oops I touched ESC key!'
```

Installation
------------
Programmed and tested on Ubuntu 16.04 and xfce.

```
$ autoreconf
$ ./configure
$ make
# make install
```

Packages you may need to install to build this program:
- build-essential
- autoconf
- libxext-dev
- libxi-dev
